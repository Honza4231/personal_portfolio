# Moje Portfolio – Jan Zima 

Zdravím! Tohle je moje osobní portfolio, které jsem nakódoval v rámci pololetní práce 4. ročníku. Je to pro mě takový "hub", kde propojuju všechno, co studuji a co mě baví – od grafického designu přes kódování až po tvorbu webů.

## O čem tenhle web je?

Je to místo, kde ukazuji své školní i vlastní projekty. Web je rozdělený na několik částí:

- **HomePage**: Úvodní stránka s informacemi o mně a rychlým přehledem.
- **WWW ((Project Gallery)**: Galerie mých webových projektů. Najdete tu například návrh e-shopu **Nutristar**, design webu pro hotel **Grammy Pica**, fanouškovskou discografii **Smack One** nebo alternativní brand **Vintage Rhymes**.
- **DIM (Gallery Dept.)**: Sekce věnovaná grafice a multimédiím. Zde prezentuji své plakáty, loga a retuše. Nově jsem přidal i **filtrování** (Illustrace, Photomanipulace, Grafické designy), takže si můžete prohlédnout přesně to, co vás zajímá.
- **About Me**: Něco málo o mně, mé cestě a dovednostech.
- **Contact**: Jednoduchý způsob, jak mě kontaktovat.

## Jak jsem na tom pracoval?

Web jsem dělal úplně od nuly ve **VS Code**. 
- Základem je **HTML5** pro sémantickou strukturu.
- Pro stylování jsem použil **CSS3**. Design je laděný do moderních tmavých tónů (Dark Charcoal) s výraznými akcenty oranžové (Vibrant Orange) a lososové (Soft Salmon). Využívám moderní techniky jako **Flexbox** a **CSS Grid**.
- Aby byl web živější, použil jsem **JavaScript** na interaktivní prvky, jako jsou filtry v galerii, modální okna pro náhledy projektů a animace.
- Celý web je **responzivní** (Mobile-First přístup), takže vypadá skvěle na mobilu, tabletu i desktopu.

Při psaní kódu mi hodně pomáhal **Copilot** přímo ve VS Code, který mi usnadnil řešení některých složitějších částí a zrychlil workflow.

## Jak dostat web na internet (GitHub & Netlify)

Aby byl web online a dostupný pro všechny, využil jsem kombinaci GitHubu a Netlify:

1.  **GitHub**: Nejdřív jsem vytvořil repozitář, inicializoval git (`git init`), přidal soubory a provedl push. To mi slouží jako záloha a verzování projektu.
2.  **Netlify**: Pro hosting jsem zvolil Netlify. Je to super jednoduché – stačilo připojit můj GitHub účet, vybrat správný repozitář a Netlify web samo nasadilo. Jakmile udělám změnu na GitHubu, web se automaticky aktualizuje.
